OSM Paraguay

Este sitio se realizó porque en la parte de descargas de GeoFabrik no esta incluído Paraguay.
Un grupo de Python Paraguay quiso disponibilizar el mapa de Paraguay. :)


Pasos realizados:

  1. Descargar de Geofabrik la parte de América del sur
  2. Ejecutar en la terminal: ./osmconvert \ /media/humitos/Seagate\ Expansion\ Drive/osm-data-extract/south-america-latest.osm.pbf \ -B=paraguay.poly \ --out-pbf \ -o=/media/humitos/Seagate\ Expansion\ Drive/osm-data-extract/south-america-paraguay.osm.pbf \ -t=/media/humitos/Seagate\ Expansion\ Drive/osm-data-extract/osmconvert_tempfile.`date +"%s"`
  3. Splitter
  4. Crear un polígono. Paraguay
  5. Cortar un .osm mediante un .poly
